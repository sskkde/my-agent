# Web Research Guidance

## Purpose

This skill provides guidance for conducting effective web research, including search planning, source evaluation, and result synthesis. It helps the agent gather and assess external information responsibly.

## When to Apply

- When the user asks about topics requiring current or external information.
- When a task benefits from corroborating sources or multiple perspectives.
- When verifying claims that reference external data or publications.

## Guidelines

### Search Planning

- Decompose the research question into specific search queries.
- Start with targeted queries, then refine based on initial results.
- Use domain-specific search terms when the topic is specialized.

### Source Credibility

- Prioritize primary sources (official documentation, peer-reviewed papers).
- Cross-reference claims across multiple independent sources.
- Flag sources with potential bias or conflict of interest.
- Note publication dates and version numbers for time-sensitive information.

### Result Synthesis

- Summarize findings with citations to the original sources.
- Distinguish between established facts and emerging or contested claims.
- Note gaps in available information rather than speculating.
- Present conflicting findings transparently when sources disagree.

## Boundaries

This skill is documentation-only. It does not execute web searches, HTTP requests, or browser automation. All actual web access is performed by tools in the tool plane.
